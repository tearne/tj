# Change: Shared Tick Row
**Type**: Proposal
**Status**: Approved

## Summary

Replaced the four per-deck tick rows (2× top, 2× bottom across both decks) with a single shared tick row positioned between the two detail waveforms, saving 3 rows overall.

The shared tick row sits at the bottom of the Deck A detail panel. Beat grids from both decks are rendered using directional braille patterns: deck A ticks point upward (tip + 3-wide base), deck B ticks point downward (3-wide base + tip), and coincident ticks form a diamond shape. Each tick mark spans two adjacent braille characters for a 3-dot-wide shape with half-column sub-pixel scrolling preserved.

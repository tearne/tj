# Proposal: Level Bar Colour
**Status: Draft**

## Intent
The level bar character in the info bar is currently styled with `DarkGray` (same as surrounding dim text), making it visually indistinct. It should use the same dark yellow as the spectrum highlight to give it a consistent accent colour that sets it apart from plain labels.

## Specification Deltas

### MODIFIED
- The level bar character (`▁▂▃▄▅▆▇█`) in the info bar is rendered in dark yellow (`Rgb(120, 100, 0)`), matching the spectrum highlight colour, rather than dim grey.

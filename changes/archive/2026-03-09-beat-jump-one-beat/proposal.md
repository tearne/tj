# Proposal: Beat Jump Unit — 1 Beat
**Status: Draft**

## Intent
The smallest beat jump unit is currently 4 beats. Adding a 1-beat unit allows fine rhythmic navigation, useful for cueing and alignment.

## Specification Deltas

### MODIFIED
- **Beat Jump**: The available beat jump units are 1, 4, 8, 16, 32, 64, and 128 beats (previously 4, 8, 16, 32, 64, 128). Key `1` selects 1 beat; keys `2`–`7` select 4, 8, 16, 32, 64, 128 beats respectively.

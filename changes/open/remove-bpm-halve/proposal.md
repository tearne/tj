# Proposal: Remove BPM Halve/Double Controls
**Status: Ready for Review**

## Intent
The `h` (halve BPM) and `H` (double BPM) controls are no longer needed. They can be removed to simplify the key binding set.

## Specification Deltas

### REMOVED
- `h` — halve BPM.
- `H` — double BPM.
- `bpm_halve` and `bpm_double` config keys.
- BPM halve/double from the `[?]` help overlay.
